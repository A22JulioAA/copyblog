HTML introduction

25/09/2022

HTML provides structure to the content appearing on a website, such as images, text, or videos. HTML stands for HyperText Markup Lenguage:

  • A markup language is a computer language that defines the structure and presentation of raw text.
  • In HTML, the computer can interpret raw text that is wrapped in HTML elements.
  • HyperText is text desplayed on a computer or device that provides acces to other text trough links, also known as hyperlinks.
